# 🧠 HEALTH AI — Co-Creation Platform

> A full-stack platform that connects **engineers** and **healthcare professionals** to build medical AI projects collaboratively.

---

## 🚀 Tech Stack

**Frontend**
- React 18 · TypeScript · Vite · TailwindCSS  
- Zustand · React Hook Form · Zod  

**Backend**
- Node.js · Express · TypeScript  
- MongoDB · Mongoose  

**Auth & Security**
- JWT Authentication  
- bcrypt  
- Helmet · Rate Limiting · Mongo Sanitize  

---

## ✨ Features

- 🔐 Secure authentication (JWT + role-based access)
- 🧩 Post lifecycle system (draft → active → completed)
- 🤝 Smart matching between users
- 📅 Meeting & collaboration workflow
- 🔔 Real-time notification system
- 🛡️ Admin panel (user moderation & logs)
- 📦 GDPR-ready (data export & account deletion)
